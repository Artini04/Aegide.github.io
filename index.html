Notice!

This project is a fork from SDM0/ Aegide. I've done some refactors within the code but most of them are still based on the original code. My aim here is to have a better mobile view. Site still WIP!

Only works with natives mons available in Pokémon Infinite Fusion v5! The data used here are from the game, but is mostly based on generation 7.

Fusion IDs in green are made by artists from the Infinite Fusion Discord and Reddit. Fusion IDs in red are from Japeal. They were extracted and fixed by Aegide.
